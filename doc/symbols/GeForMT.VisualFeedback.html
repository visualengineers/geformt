<private> <inner> _createFeedbackEnvironment()
Create two canvas elements on top of the web content to draw visual feedback on it. One canvas for updating current position of contacts, the other canvas for drawing gesture pathes on it. Requirements for this feature: 1. This feature requires HTML5. 2. The browser must support the CSS property 'pointer-events'.
