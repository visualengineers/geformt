<inner> GeForMT.Observation-Contact(identifier, screenX, screenY, clientX, clientY, pageX, pageY, radiusX, radiusY, rotationAngle, force)
Contact object that generalizes a single mouse or touch contact point. The object interface is based on the Touch Event API Recommendation of W3C.
Parameters:
{String} identifier
{Number} screenX
{Number} screenY
{Number} clientX
{Number} clientY
{Number} pageX
{Number} pageY
{Number} radiusX
{Number} radiusY
{Number} rotationAngle
{Float} force
See:
http://www.w3.org/TR/2011/CR-touch-events-20111215/
https://developer.mozilla.org/en/DOM/Touch
