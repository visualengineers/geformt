<inner> {Object} GestureRecognitionEventListener
Holds a list for a specific event of this module with callback methods of listeners. Listeners must register for these events via API method. Possible event types: 'recognitionstarted': Event if recognition processor begins to process input data.; 'recognitionfailed': Event if recognition processor couldn't recognize any gesture.; 'gesturerecognized': Event if gesture has been recognized.; 'candidatesrecognized': Event if gesture has been recognized but further gestures are expected.

<inner> GestureRecognizedEvent(identifier, model, eventProfileHistory, gesturePathes)
Event interface gesturerecognized event.
Parameters:
{String} identifier
Identifier of the gesture
{GeForMT.GestureModel.Gesture} model
{Array} eventProfileHistory
Nested lists of all events representing the gesture. Structure: The list contains gesture sequences. Every sequence represents a sequentially performed gesture. A sequence is a list of events that occured while performing the gesture.
{Array} gesturePathes
Nested list of templates representing the gesture. Structure: The list contains gesture sequences. Every sequence represents a sequentially performed gesture. A sequence is a list of template objects (type GeForMT.TemplateBuilder.Template). Every template represents a single stroke. More than one template in the sequence means, that a Multitouch gesture has been performed.
